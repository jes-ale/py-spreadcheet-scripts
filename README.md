## Requirements
1. Python 3:
   ```bash
   python3 --version
   ```

## Installation
1. Create a virtual environment (optional but recommended):
   ```bash
   python -m venv venv
   ```
2. Activate the virtual environment:
   - On Windows:
     ```bash
     venv\Scripts\activate
     ```
   - On macOS and Linux:
     ```bash
     source venv/bin/activate
     ```
3. Install the project dependencies:
   ```bash
   pip install -r requirements.txt
   ```
